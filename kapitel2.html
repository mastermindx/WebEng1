Kapitel 2

Starter Kit: Tools für die Website in der Cloud

In diesem Kapitel ging es grundsätzlich um den Umgang mit den Tools welche während des MOOCs eingesetzt wurden bzw. werden. Zum einen gab es Teilnehmer die keinerlei Schwierigkeiten mit dem Umgang hatten, zum anderen waren einige etwas länger mit der Einarbeitung beschäftigt. Die Aufgabe dieses Kapitels bestand darin eine Seite, welches mittels Bootstrap erstellt wurde, in die Cloud...hier CloudControl...zu bringen. Die Versionierung wurde mittels GitHub durchgeführt.

Der Reader zu Kapitel 2 beinhaltet folgende Themengebiete:

  • Virtuelle Maschine
  • VirtualBox
  • BitTorrent
  • Debian
  • Apache HTTP Server
  • Eclipse (IDE)
  • Git
  • Bootstrap (Framework)
  • Cloud-Computing
  • Mozilla Firefox
